◆ am_hal_burst_mode_initialize()

uint32_t am_hal_burst_mode_initialize ( am_hal_burst_avail_e * peBurstAvail)

Burst mode initialization function.

Parameters
peBurstAvail- Availibility of feature

This function initializes the Apollo3 MCU for Burst Mode operation. It does not set the MCU into Burst Mode. This should be called once at system initialization if Burst Mode is going to be used in the system.

Returns
status of API call.
