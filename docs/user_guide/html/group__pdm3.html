◆ AM_HAL_PDM_DMA_THRESHOLD_MIN

#define AM_HAL_PDM_DMA_THRESHOLD_MIN   16

DMA threshold minimum.

The PDM DMA works best if its threshold value is set to a multiple of 4 between 16 and 24, but it will technically allow threshold settings between 4 and 24. This macro sets the minimum threshold value that the HAL layer will allow.

◆ am_hal_pdm_status_e

enum am_hal_pdm_status_e

PDM-specific error conditions.

Enumerator
AM_HAL_PDM_STATUS_BAD_TOTALCOUNT 

The PDM HAL will throw this error if it can't find a threshold value to match the total-count value passed in by a caller requesting a DMA transfer. The PDM hardware requires all DMA transactions to be evenly divisible in chunks of one FIFO size or smaller. Try changing your ui32TotalCount value to a more evenly divisible number.
