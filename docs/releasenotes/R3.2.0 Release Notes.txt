**********************************************************************************************

This is SDK update targeting the Apollo3 family of MCUs include Apollo3, Apollo3 Blue and
Apollo3 Blue Plus.  Released on 05Sep2024 (revision release_sdk_3_2_0-dd5f40c14b).

**********************************************************************************************

SDK supports all Apollo3 evaluation boards from Ambiq.
- Apollo3 Blue EVB (see /boards/apollo3_evb)
- Apollo3 Blue EVB + Cygnus Shield (see /boards/apollo3_evb_cygnus)
- Apollo3 Blue Plus EVB (see /boards/apollo3p_evb)
- Apollo3 Blue Plus EVB + Cygnus Shield (see /boards/apollo3p_evb_cygnus)

SDK includes a number of predefined examples for each configuration.

IDE/ToolChain Support:
----------------------
AmbiqSuite SDK supports GCC, IAR , Segger and Keil IDEs.
Support for older Keil/ARMCC 5060750 has been deprecated.

This release has been tested with:
- GCC 13.2.1 20231009 [Arm GNU Toolchain 13.2.Rel1 based on GCC 13.2]
- IAR ANSI C/C++ Compiler V9.40.2.374/W64 for ARM
- ARMCC 6190004 [Keil V5.38a using ArmClang V6.19]
- GCC SEGGER-CC 16.0.9 [Embedded Studio v8.10d released 2024-03-25]

* All devices are natively supported in J-Link v6.80a or later.

* A .vscode folder with a sample configuration is included in the hello_world and freertos_mspi_display examples, setup for apollo3p.
  A Visual Studio Code quickstart guide is provided to help get the prepackaged configuration running on your environment. Please see 
  the Visual Studio Code Quickstart Guide in /docs/Apollo3 Family AmbiqSuite Visual Studio Code Project Example Quick Start Guide v1p0.pdf.
