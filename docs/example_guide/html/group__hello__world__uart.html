Hello World - UART Example
Examples

Detailed Description

This example prints a "Hello World" message with some device info over UART at 115200 baud. To see the output of this program, run a terminal appl such as Tera Term or PuTTY, and configure the console for UART. The example sleeps after it is done printing.
