Cache Performance Example
Examples

Detailed Description

Purpose: This example provides a demonstration of the cache monitor to check the cache hit rate and cache miss number.

Additional Information: If the fireball device card is used, this example can work on: Apollo3p_evb + Cygnus Recommend to use 3.3V power supply voltage.
