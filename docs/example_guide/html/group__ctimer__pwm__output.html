CTIMER PWM Example
Examples

Detailed Description

Purpose: This example shows one way to vary the brightness of an LED using a timer in PWM mode. The timer can be clocked from either the XTAL (default) or the LFRC, selectable by a define, USE_XTAL.

Printing takes place over the ITM at 1M Baud.
